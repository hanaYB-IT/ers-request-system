# IELTS Request System

## 📌 Overview
The IELTS equivalency process at universities is often complex, 
time-consuming, and prone to errors due to manual handling and unclear procedures.
Students frequently submit incomplete or incorrect requests,
while staff spend significant time verifying data and managing applications across multiple steps.

This project introduces an automated IELTS Equivalency Request System that simplifies the entire workflow. 
It provides clear guidance for students, automatic score validation, and an efficient approval process for staff and
administrators, reducing errors and improving overall efficiency.

## 🚀 Features
- Automated IELTS score validation  
- Request submission and tracking  
- Staff verification workflow  
- Admin approval system
